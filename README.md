# 驾驶员状态检测

[Distracted Driver Detection](https://www.kaggle.com/c/state-farm-distracted-driver-detection)

![](driver.gif)


## 描述

使用深度学习方法检测驾驶员的状态。

* 输入：一张彩色图片
* 输出：十种状态的概率

状态列表：

* c0: 安全驾驶
* c1: 右手打字
* c2: 右手打电话
* c3: 左手打字
* c4: 左手打电话
* c5: 调收音机
* c6: 喝饮料
* c7: 拿后面的东西
* c8: 整理头发和化妆
* c9: 和其他乘客说话
